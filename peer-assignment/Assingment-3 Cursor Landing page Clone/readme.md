# Assignment 3 - Cursor Landing Page Clone

A static HTML/CSS clone of the Cursor landing page. This project focuses on layout, typography, and responsive sectioning for a marketing-style homepage.

[Preview Link](https://heartfelt-cajeta-00ec09.netlify.app/)

![Preview](https://h.kuntalworks.com/image_cdn/uploads/upload_6983189173d5d5.21848681.webp)

## Features
- Navigation bar with brand logo and CTA buttons
- Hero section with primary call-to-action
- Trusted-by logo strip
- Reusable feature cards and testimonial layout
- Changelog and highlights sections
- Footer with multi-column links and controls

## Tech Stack
- HTML5
- CSS3 (custom styles in `css/styles.css`)

## How to Run
1. Open `index.html` in any modern browser.

## Project Structure
- `index.html`
- `css/`
- `images/`
- `fonts/`

## Notes
- Some logos and avatars are loaded from external URLs.
